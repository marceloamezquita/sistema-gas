# Sistema Gas

## Project Overview

Sistema Gas is a comprehensive solution designed to manage gas distribution and sales efficiently. This project aims to streamline operations, enhance customer service, and ensure the accurate tracking of gas inventories.

### Features
- **Inventory Management**: Track gas cylinders and their availability in real-time.
- **Order Processing**: Facilitate online orders with an intuitive user interface.
- **Customer Management**: Maintain customer records and transaction histories.
- **Reporting**: Generate reports to analyze sales and inventory trends.

### Technologies Used
- **Frontend**: React, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB

To get started with the project, clone the repository and install the required dependencies.
